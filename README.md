# Remindly
<br>

**Remindly** is a simple Java desktop app to manage homework and tasks. Built with **JavaFX** and **PostgreSQL**, it lets you add, view, and complete tasks with due dates


<br>
<br>


## 🚀 Features
- Add/view/delete tasks
- Set due dates
- Mark tasks as done
- Data stored in PostgreSQL



## 🛠 Tech Stack

| Component       | Technology          |
|-----------------|---------------------|
| Language        | Java 17+            |
| GUI             | JavaFX              |
| Database        | PostgreSQL          |
| Build Tool      | Maven               |
| IDE (suggested) | IntelliJ IDEA       |
